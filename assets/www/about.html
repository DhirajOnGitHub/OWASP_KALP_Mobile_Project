OWASP KALP Mobile Project

OWASP KALP Mobile is an android application developed for users to view OWASP Top 10 (WEB and MOBILE) on mobile device.

Introduction

OWASP KALP Mobile Project is for OWASP users around the world who want to view the OWASP Top 10 vulnerabilities, download the Top 10 list on their mobile device, and email it. This is a lightweight information of OWASP Top 10 list, Cheat Sheets as well as Prevention Cheat Sheets created from the OWASP site.

Description

OWASP KALP Mobile Project is for OWASP users around the world who want to access the Top Ten vulnerabilities on the go (on their mobile), download the Top Ten and Email it. This is light weight information of OWASP Top Ten. Any new additions to cheat sheets and prevention cheat sheets will automatically accessible on the mobile app.

Any new additions to the Top Ten list, cheat sheets and prevention cheat sheets will be accessible on the mobile application as well.

Key Features:

  • User friendly Mobile Application for Android Users.
  • Easy to access all OWASP Top 10 list related to Mobile / Web with respective year.
  • User can Download and/or Email OWASP Top 10 list in PDF format.
  • Contents are fetched from OWASP site with regular updates.

What is OWASP KALP MOBILE Project?

  1. This is an android mobile application which users can install on their android mobile/tablet.
  2. This application will list all OWASP Top 10 (2013, 2010, 2007, 2004, 2003) and OWASP Mobile Top 10 (2014 , 2012).
  3. User can view OWASP Top 10, description on each vulnerability, cheat sheets and prevention cheat sheets.
  4. For additional contents users will be redirected to OWASP site.
  5. Users can also download the selected OWASP Top 10 list in PDF format on his / her mobile and /or Email it.

The contents and description on each vulnerability, categories etc. will not be changed and same contents from OWASP site will be used.
